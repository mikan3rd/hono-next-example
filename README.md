# :rocket: hono-next-example

## Tech Stack

- Common
  - [Bun](https://bun.sh/)
  - [Biome](https://biomejs.dev/)
- Backend
  - [Hono](https://hono.dev/)

## Setup

- Install package manager **asdf**

```bash
brew install asdf
```

- Install **Bun** by asdf

```bash
asdf install
```

- Install dependencies

```bash
bun install
```
